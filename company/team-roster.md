# Team Roster

Use functional role headers in Specialist Mode. Do not create persistent persona names.

## Roles

- Founder / Chief of Staff: scope, project state, queue ranking, claim boundaries, closeout.
- Quant Researcher: target definitions, baseline model choices, validation protocol, result interpretation.
- Data Engineer: public-data ingestion, caching, manifests, schema checks.
- Research Engineer: Python package, CLI workflows, tests, reproducibility.
- Replication QA: leakage checks, reproducibility smoke tests, README/report honesty review.

## Current Depth

Minimal Specialist Layer. Role memory files, integration logs, and heavier review artifacts are intentionally skipped until the project earns them.
